 Setup Instructions

1. Download or Clone
```bash
git clone https://github.com/yourusername/FEEC-Profile.git
````

2. Open Locally

* Simply open `index.html` in your preferred browser
* No backend or database setup is required (static HTML/CSS/JS)

3. Dependencies

* [Bootstrap 5](https://getbootstrap.com/)
* [Google Fonts](https://fonts.google.com/)
* [AOS - Animate on Scroll](https://michalsnik.github.io/aos/)
* [Swiper JS](https://swiperjs.com/) (for sliders if needed)
* [FancyBox](https://fancyapps.com/fancybox/) (for gallery popups)

4. Customizations

* Replace placeholder images in the gallery with actual product/service images
* Update organizational chart, project locations, and team info as needed
* Modify `styles.css` and `script.js` for custom styling and interactivity
